# fresh_fish
A Fish restaurant ordering app made in Flutter . 

What we used in this app :

- Provider package . 

- Firebase Authentication .

- Firebase cloud firestore . 

- Cloud Messaging .

- Cloud Functions .

- Google & Facebook SignIn .

- Shared Preferences .

- Stream & Future Builders .
